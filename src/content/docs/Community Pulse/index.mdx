Every Monday this site automatically collects what happened in the macOS plus Microsoft Intune world during the previous week:

- **Microsoft announcements** from the Intune blogs on Tech Community
- **Community blog posts** from macOS admin bloggers (see the source list below)
- **Reddit highlights** from r/Intune and r/macsysadmin

An AI pipeline filters for macOS relevance and writes the short neutral summaries; a human reviews every digest before it is published. Summaries can contain mistakes - always read the linked original before acting on anything.

## Your blog is missing?

The source list lives in [config/sources.json](https://github.com/ugurkocde/intunemacadmins/blob/main/config/sources.json). Open a pull request or an issue to add (or remove) a blog.
