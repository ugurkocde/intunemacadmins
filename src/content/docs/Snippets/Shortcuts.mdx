## macOS Cloud Recovery

Intel-based Mac:
<Steps>
1. Restart resp. start the Mac.
2. Hold the following keys until you see the startup screen: Option + Command + R
3. Perform recovery.
</Steps>

Mac with Apple silicon:
<Steps>
1. Restart resp. start the Mac.
2. Press and hold the power button for a few seconds until you see "Loading startup options".
3. Perform recovery.
</Steps>
