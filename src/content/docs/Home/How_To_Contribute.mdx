We welcome contributions from the community to help improve the content on IntuneMacAdmins. Whether it's a new guide, a script, or an insight, your contribution is valuable.

<Aside type="tip">
  The easiest way is to use the "Edit page" button that you can find on the
  bottom of every page. If you edit more than one page, we recommend using forks
  and create a pull request using the method described below.
</Aside>

### Steps to Contribute

<Steps>
  <ol>
    <li>
      <strong>Fork the Repository</strong>: Start by forking the <a href="https://github.com/ugurkocde/IntuneMacAdmins">IntuneMacAdmins GitHub repository</a>.
    </li>
    <li>
      <strong>Clone the Repository</strong>: Clone your forked repository to your local machine.
      <pre><code>git clone https://github.com/your-username/IntuneMacAdmins.git</code></pre>
    </li>
    <li>
      <strong>Create a New Branch</strong>: Create a new branch for your changes. 
      <pre><code>git checkout -b my-new-branch</code></pre>
    </li>
    <li>
      <strong>Make Your Changes</strong>: Make the necessary changes or additions to the content.
    </li>
    <li>
      <strong>Commit Your Changes</strong>: Commit your changes with a descriptive message. 
      <pre><code>git commit -m "Add new guide on XYZ"</code></pre>
    </li>
    <li>
      <strong>Push Your Changes</strong>: Push your changes to your forked repository.
      <pre><code>git push origin my-new-branch</code></pre>
    </li>
    <li>
      <strong>Create a Pull Request</strong>: Go to the original repository and create a pull request from your forked repository.
    </li>
  </ol>
</Steps>
